# PaperGrid – Public Website Homepage

PaperGrid is a clean and responsive public website homepage designed using HTML and CSS.
The project focuses on layout structure, visual hierarchy, and modern UI styling to create a simple yet effective landing page experience.

Features :

  Responsive homepage layout
  
  Clean and minimal UI design
  
  Semantic HTML structure
  
  Modern CSS styling
  
  Lightweight and fast loading
  

 Technologies Used:
 
  HTML5 – Page structure and content
  
  CSS3 – Styling, layout, and responsiveness
  

 Purpose of the Project :
 
    This project was created to:
    
    Practice building real-world website layouts
    
    Improve HTML & CSS fundamentals
    
    Understand responsive design principles

    
    Create a visually appealing public homepage
